# Axiom - AFinancial Reality Engine & Market Autopsy Suite

This website has been constructed using HTML, CSS, and JavaScript technologies and deployed through Vercel platform.Key Goals of this project is overcome the knowledge barrier of finance by analyzing the fundamentals of market behavior.
The Behavior Analysis is centered on the study of the psychological aspects of markets coupled with simulation of past crises instead of trading.

 This website offers the "Financial Reality Engine," which not only simulates but also provides deeper insights on the mechanics of wealth creation.
It has an Autopsy system- giving info on complex info- the Sharpe Ratio, Annualized Volatility, Disposition Effect, or the Win Rate.  

First-Principles Approach: All financial concepts are distilled into their purest mathematical and behavioral form.
We have went forward with a minimalist UI Design which is terminal-themed, offering a low-latency interface for extensive "lock-in" learning sessions. The website has been constructed with an emphasis on web architecture and execution, mirroring the shift in the user towards technical expertise.
